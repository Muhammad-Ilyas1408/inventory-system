# Inventory & Order Management System

# Project Requirements

## Project Overview

The Inventory & Order Management System is a command-line application developed in Python that helps a small business manage products, customers, inventory, orders, invoices, and reports.

The system follows a layered architecture to separate user interaction, business logic, data models, and storage.

---

# Objectives

The project aims to demonstrate:

- Object-Oriented Programming
- Modular Design
- Layered Architecture
- File Handling
- Exception Handling
- Data Validation
- Business Logic
- Clean Code Practices
