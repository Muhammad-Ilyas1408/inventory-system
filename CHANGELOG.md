# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-07-20

### Added

- Layered architecture with domain models, services, JSON storage, and a
  modular CLI.
- Product, customer, order, and invoice CRUD workflows.
- Order workflow with customer and product selection.
- Inventory automation with insufficient-stock validation.
- Automatic invoice generation from successful orders.
- Manual invoice creation from existing orders and duplicate invoice prevention.
- Inventory, sales, order, customer, and invoice reporting.
- Comprehensive automated test coverage.

### Changed

- Polished CLI workflows, headings, confirmations, and error presentation.
- Improved automated test coverage across business workflows.
- Updated repository documentation for the v1.0.0 release.
